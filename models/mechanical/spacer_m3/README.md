# M3 Spacer

Simple spacer for M3 hardware with a clearance hole and configurable height.

## Parameters

- `outer_d` – outer diameter (default 7 mm)
- `hole_d` – clearance hole diameter (default 3.2 mm)
- `height` – spacer height (default 10 mm)
- `$fn` – cylinder smoothness (default 128)

## Render

Use OpenSCAD GUI or run `scripts/render_example.sh` to generate an STL via CLI.
